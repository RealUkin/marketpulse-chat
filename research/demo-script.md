# MarketPulse Chat — Demo Video Script (~3 min)

For the Loom/YouTube submission. **Hook-first, show don't tell, real data, end on the repo.** Judges are crypto/streamer personalities (Banks, Ansem, Polymarket) — keep it fast, confident, "this is worth $10k" energy.

**Before recording:** run `npm run dev`, open `localhost:3000`. Pre-connect a busy live channel (e.g. a big Twitch streamer) via the Connect modal so real chat is flowing — OR leave Demo Mode on (it's lively + reliable). Have the OBS overlay (`/overlay`) ready in a second tab or actually inside OBS for the finale.

---

### 0:00–0:18 · The hook (open ON the product)
- Screen: the live dashboard, messages pouring in from all 4 platforms.
- Say: *"This is MarketPulse Chat. Twitch, X, Kick, and YouTube — every chat, one real-time feed. Built for streamers who go live everywhere at once."*

### 0:18–0:45 · Unified feed + real emotes + badges
- Point at the per-platform colored avatars/labels, the sub/mod/VIP badges, and real emotes rendering.
- Say: *"Every message is labeled by platform, with normalized sub, mod, and VIP badges — and full emotes: 7TV, BetterTTV, FrankerFaceZ, channel sub emotes, YouTube emojis. The stuff your chat actually uses."*

### 0:45–1:10 · Connect = no login (the Restream killer)
- Click **Connect platforms**. Toggle platforms on, type a channel, Connect.
- Say: *"Connect your platforms in one click. And here's the kicker — no login needed to read chat. Restream makes you hook up accounts before you see anything. We don't."*

### 1:10–1:35 · Events pop (subs / bits / raids)
- A sub / bits / raid / super-chat banner appears (Demo Mode triggers these).
- Say: *"Subs, gifted subs, bits, raids, super-chats — they pop right in the feed, across every platform, so you never miss the moment someone supports you."*

### 1:35–2:05 · Crypto Hype Intelligence (optional)
- Toggle **📊 Hype** on. Point at $tickers, sentiment, top tickers, live Polymarket odds.
- Say: *"For trading streams: live $ticker detection, chat sentiment, and real Polymarket odds right next to chat. And it's optional — one toggle and it's a clean chat for any streamer."*

### 2:05–2:25 · Scam defense (the unique one)
- Point at a ⚠ flagged scam message + the Safety panel.
- Say: *"Crypto chats get flooded with scam links and wallet drainers. We flag them automatically — nobody else does this."*

### 2:25–2:55 · OBS overlay + feature-on-stream (the finale)
- Show the transparent `/overlay` (ideally in OBS). Star a message in the dashboard → the **Featured banner** pops on the overlay. Emotes "pop."
- Say: *"Drop the overlay into OBS — transparent, emote pops, all your chats live. And star any message to feature it on stream for your viewers."*

### 2:55–3:05 · Close
- Cut to the GitHub repo (clean README).
- Say: *"Existing tools merge chat. MarketPulse helps crypto streamers understand it. Repo's in the description — let's go."*

---

## Delivery tips
- Talk over live action the whole time — never a static slide.
- If a live channel is quiet, Demo Mode is your safety net (varied chat + events + scams on tap).
- Keep cuts tight; energy high. ~3 minutes max.
- End frames: the overlay featured banner, then the GitHub page.
